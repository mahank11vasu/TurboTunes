# TurboTunes

TurboTunes is a vehicle servicing platform that allows users to book services, request part replacements, and manage workflow dynamically. It includes features for both customers and mechanics with a responsive user interface and admin panel for management.

## Features
- User login and account creation.
- Book vehicle services and replacement parts.
- Mechanic workflow management.
- Admin panel for managing bookings, services, and users.
- Responsive design for mobile and desktop.

## Technologies Used
- **Frontend:** React, Bootstrap
- **Backend:** PHP
- **Database:** MySQL
